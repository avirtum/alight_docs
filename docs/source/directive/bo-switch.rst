bo-switch
=========

Overview
--------

* Command switch. **Bind once**.
* Input argument: expression

.. code-block:: html
    :caption: Example

    <div bo-switch="value">
        <div bo-switch-when="one"></div>
        <div bo-switch-when="two"></div>
        <div bo-switch-default></div>
    </div>
